WwDraw - A time-based drawing library for capturing and replaying multi-layered, multi-author drawings.

npm Try It

Features

  • Real-time drawing capture
  • Captures the full drawing performance
  • Multiple brushes and colors
  • Supports multiple layers
  • Supports multiple authors (artists)
  • Pixijs and Canvas support
  • In-browser and Electron support
  • JSON file format for captured drawings
